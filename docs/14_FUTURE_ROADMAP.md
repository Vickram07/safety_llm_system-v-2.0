# Future Roadmap

## Summary of Missing Modules (To be integrated)

1.  **RFID-bound Personal Identity**: Hardware + Database.
2.  **Private Encrypted Sessions**: End-to-end encryption for operator chat.
3.  **Hospitality Mode**: A "Peace Time" mode where logic is relaxed for guest services.
4.  **Continuous Evaluation Loop**: Infinite loop logic (currently single-shot).
5.  **Multi-person Routing**: Complex graph algorithms.
6.  **Evidence-Based Reporting**: Auto-citation of sensor IDs in every log.
7.  **Lost RFID Fallback**: Vision-based re-identification.
8.  **Autonomous Infra Protection**: PLC integration.
9.  **Critical Room Logic**: Specific rules for Server Rooms/Bio-Labs.
10. **Rail-Based Sprinklers**: Hardware + Targeting logic.
11. **Zero-Trust Sensors**: Crypto-signed MQTT packets.
12. **Sealed Commands**: Digital signatures for actuators.
13. **Anti-Spoofing**: Vision/Sensor adversarial defense.
14. **Air-Gapped Mode**: Hardware switch + local mesh network.
15. **Immutable Ledger**: Blockchain container.
16. **Proof-of-Life**: mmWave/CO2 integration.
17. **Uncertainty Engine**: Probabilistic programming layer.
18. **Moral Pause**: Ethical threshold logic.
19. **Panic Prediction**: Crowd simulation integration.
20. **Offline Learning**: CI/CD pipeline for model fine-tuning.
